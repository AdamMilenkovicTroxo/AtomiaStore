Atomia Store is a web application and framework for ordering products and services via the Atomia platform. It can be used as a stand-alone web shop (the default), or parts of Atomia Store can be used to implement an embedded store in other applications.

Getting Started With a Custom Theme
===================================

1. Clone the AtomiaStore repository
2. Open PowerShell and run `AtomiaStore\ThemeStarter\startnewtheme.ps1 <themeName>`
3. Then head over to [documentation.atomia.com](http://documentation.atomia.com) and read about implementing your theme.
